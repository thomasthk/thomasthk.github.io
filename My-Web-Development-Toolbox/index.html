🧰 My Web Development Toolbox

Thomas, 
February 1st, 2020 · 1 min read

Text Editor:

VS Code is one of my favorite code editor, it is customizable, features rich and have lots of extensions. You can easily build your own code editor with your own settings.

Visual Studio Code

Static Site Generator:

When I first heard of the term JAMStack, I am already fascinated by it. I like the idea that it is simple, fast and secure.

I just want to build a simple personal site. I don’t need a database, I don’t need to manage my web server configs and settings, I don’t need to learn the complexity of using a features rich CMS application.

JAMStack explained why to consider the JAMStack

For choosing a static site generator, it’s all just a matter of taste and I recommend you to try some of them before making your own decision.

StaticGen lists a number of open source static site generators for you to compare more easily.

I finally settled myself with the GatsbyJS. If you have difficulty in making decisions, may be you can consider to start with GatsbyJS too. It has well written tutorials, tons of plugins and a strong and active community.

GatsbyJS

Website theme:

LekoArts provides high quality GatsbyJS theme for starters and beginners. This 1st version of this blog is based on the minimal blog theme from LekoArts. So that I can focus on the content of the site and then further design and fine tune the layout in the future.

LekoArts

For this version of the blog, it is based on the Novela theme.

Novela

Static Web Host:

Github Pages can be set up so easily from the Github repository. With just a click in the setting, you created a personal portfolio site for free.

Github Pages

Version Control:

Github is now free for everyone, you can create unlimited public/private repositories with it. If you still haven’t use a version control service for your own project, I recommend you to act now and start with Github.

Github
